# Button 按钮组件(内置防抖)

:::tip tips
继承el-button所有属性 <br/>新增 `time` 属性（多少时间内点击；默认 1 秒）
:::

### 基础用法

:::demo
Button/base
:::

### Attributes

> 继承所有 ElButton 的属性。<br/>
> 详情可查看 [-> Element Plus Button](https://element-plus.org/zh-CN/component/button.html#button-%E5%B1%9E%E6%80%A7)

| 参数 | 说明     | 类型   | 默认值(ms) |
| :--- | :------- | :----- | :--------- |
| time | 防抖时间 | number | 1000       |
